# Scalar社 エンタープライズセールス・オペレーションマニュアル

## 1. 本マニュアルの目的

このマニュアルは、Scalar社のエンタープライズセールスチームの活動を標準化し、効率性と生産性を最大化することを目的とする、**唯一の正しい情報源 (Single Source of Truth)** です。

全員が同じルールとプロセスを共有することで、以下の実現を目指します。

- **新メンバーの迅速なオンボーディング**
- **営業活動の質の均一化と向上**
- **CRMへのデータ入力徹底による、データドリブンな営業戦略の立案**
- **見積もりや契約プロセスの迅速化**

> [!IMPORTANT]
> 不明な点があれば、まずこのマニュアルの該当項目を参照してください。マニュアルに記載がない、または解釈に迷う点については、Sales Opsに確認してください。

## 4. マニュアルの更新について

このマニュアルは生きたドキュメントです。市場の変化や営業活動で得られた新たな知見を反映し、常に最新の状態に保つ必要があります。
内容の修正や追加の提案がある場合は、**GitHubのIssueを立てるか、Pull Requestを作成してください。** 定期的にセールスオペレーションチームがレビューし、内容を更新します。
